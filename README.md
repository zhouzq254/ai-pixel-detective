# 🕵️‍♂️ AI像素侦探社

**全球首个AI生成的侦探推理游戏**

## 🎮 游戏简介

《AI像素侦探社》是一款结合复古像素风格与现代AI技术的侦探推理游戏。每个案件都由DeepSeek AI动态生成，每次游戏体验都独一无二！

### ✨ 核心特色

- **🤖 AI生成案件** - 每次游戏案件剧情、嫌疑人、线索都不同
- **🎭 智能嫌疑人** - 每个NPC都有独特性格，会撒谎、隐瞒、合作
- **🕹️ 复古像素风** - 90年代经典像素风格，怀旧情怀
- **🔍 真实推理体验** - 调查现场、收集证据、分析线索、指认凶手
- **📱 H5网页游戏** - 无需下载，打开即玩，支持移动端

## 🚀 技术架构

### 技术栈
- **游戏引擎**: Phaser.js 3.60
- **编程语言**: TypeScript
- **AI服务**: DeepSeek API
- **构建工具**: Vite
- **部署平台**: Cloudflare Pages

### 项目结构
```
ai-pixel-detective/
├── src/
│   ├── game/           # 游戏核心逻辑
│   ├── scenes/         # Phaser游戏场景
│   ├── ai/             # DeepSeek AI集成
│   ├── data/           # 游戏数据模型
│   ├── ui/             # 界面组件
│   └── utils/          # 工具函数
├── assets/             # 游戏资源
│   ├── images/         # 像素图片
│   ├── audio/          # 音效音乐
│   └── fonts/          # 字体
├── public/             # 静态资源
└── docs/               # 设计文档
```

## 🎯 游戏玩法

### 核心循环
```
进入案件 → 调查现场 → 收集证据 → 询问嫌疑人 → 
分析线索 → 推理矛盾 → 指认凶手 → 案件解决 → 
获得评价 → 解锁新案件
```

### 主要系统
1. **调查系统** - 点击场景收集证据
2. **对话系统** - 与AI嫌疑人互动
3. **推理系统** - 在侦探板上分析线索
4. **评价系统** - 根据破案效率评分

## 🔧 开发指南

### 环境要求

- npm 9+

```bash
npm install
```

### 开发运行
```bash
npm run dev
```

### 构建部署
```bash

```

1. 复制 `.env.example` 为 `.env`
2. 添加你的DeepSeek API Key
3. 配置游戏参数

## 🤖 AI集成

### DeepSeek API使用
游戏使用DeepSeek API实现以下功能：
- **案件生成** - 动态生成案件剧情
- **嫌疑人对话** - 实时生成对话内容
- **线索生成** - 创建案件线索和证据
- **助手提示** - 提供破案提示

### API成本控制
- 对话缓存机制
- 案件预生成

## 🎨 美术资源

### 像素风格规范
- **分辨率**: 320x240（内部），640x480（显示）
- **色板**: 16色NES风格调色板
- **角色**: 16x16或32x32像素小人
- **动画**: 2-3帧简单动画

### 资源来源
- OpenGameArt.org - 免费像素素材
- itch.io - 免费UI组件
- Freesound.org - 音效资源
- 自制像素画
